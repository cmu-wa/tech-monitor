ربط Google Sheet
اجعل الأخبار مشتركة بين جميع الزوار عبر Google Sheets كقاعدة بيانات مجانية


            

                

                    
كيف تعمل هذه الميزة؟

                    

                        
1. تنشئ جدول بيانات في Google Sheets وتضيف فيه الأخبار كـ JSON.

                        
2. تنشر الجدول كملف JSON عبر Google Apps Script.

                        
3. تضع رابط النشر هنا — فيقرأه الموقع ويعرض البيانات لجميع الزوار.

                        
4. عند إضافة خبر جديد من هنا، يُحفظ محلياً وتُصدّره لنسخه يدوياً إلى Google Sheets.

                    

                


                

                    
                    
                


                

                    
                    
                    
                


                


                

                    
دليل الإعداد خطوة بخطوة

                    

                        
الخطوة 1: إنشاء Google Sheet

                        
اذهب إلى sheets.google.com وأنشئ جدولاً جديداً. في الخلية A1 اكتب: data ثم في الخلية B1 الصق مصفوفة JSON الكاملة للأخبار.

                        
الخطوة 2: إنشاء Apps Script

                        
من القائمة: Extensions → Apps Script. الصق الكود التالي:

                        
function doGet() {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = sheet.getRange("B1").getValue();
  return ContentService.createTextOutput(data)
    .setMimeType(ContentService.MimeType.JSON);
}

                        
الخطوة 3: النشر

                        
اضغط Deploy → New deployment → Type: Web app. اختر "Anyone" ثم Deploy. انسخ رابط الـ URL والصقه أعلاه.

                        
الخطوة 4: تحديث البيانات

                        
عند إضافة خبر جديد من هذه اللوحة، اضغط "Generate JSON" من أدوات البيانات، ثم انسخ المحتوى والصقه في الخلية B1 في Google Sheet.
