أعدادات الاتصال بمستودع GitHub لنشر الأخبار مباشرة بضغطة زر

كيف تحصل على Personal Access Token؟

1. اذهب إلى github.com/settings/tokens

2. اضغط "Generate new token (classic)"

3. أعطه اسماً مثل: tech-monitor

4. اختر الصلاحية: repo (كاملة)

5. اضغط Generate وانسخ التوكن

الصيغة: اسمالمستخدم/اسمالمستودع

كيف يعمل النشر؟

1. تضيف/تعديل الأخبار من لوحة التحكم (محلياً)

2. تضغط زر "نشر الآن" من لوحة المعلومات

3. البرنامج يرفع ملف data.json إلى مستودع GitHub تلقائياً

4. الصفحة الرئيسية تقرأ data.json وتعرض الأخبار للجميع
